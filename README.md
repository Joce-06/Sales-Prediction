
# 📈 Sales Prediction using Advertising Data

##  Overview
This project predicts product sales based on advertising spend across TV, Radio, and Newspaper.

##  Tech Used
- Python
- Pandas, NumPy
- Scikit-learn
- Seaborn, Matplotlib

##  Steps
1. Data Exploration
2. Visualization
3. Model Building (Linear Regression)
4. Evaluation

## Results
- R² Score: ~0.9
- Model shows strong correlation between TV/Radio ads and sales

##  Insights
- TV advertising has highest impact
- Radio also contributes significantly
- Newspaper has weaker influence
